**CREAR LA BD LARAVEL **
** Comandos **  
1. composer install
2. copy env.example ->  env
3. php artisan key:generate
4. php artisan migrate:fresh --seed
5. npm install
6. npm run dev
7. php artisan storage:link
8. php artisan serve
